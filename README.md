Features
 Draw using your index finger when you close your hand (excluding the index finger).
 
 Clear the canvas by moving your finger to the top-left corner (within a small zone).
 
 Pygame interface shows the real-time feed and drawing canvas.
 
 Draws with a yellow color by default.
 
| Action                     | Gesture                                        |
| -------------------------- | ---------------------------------------------- |
| **Start Drawing**          | Index finger up, other fingers folded          |
| **Stop Drawing**           | Open hand or all fingers up                    |
| **Clear Canvas**           | Move index finger to top-left (x < 50, y < 50) |
| **Force Clear (Keyboard)** | Press `C` key                                  |
